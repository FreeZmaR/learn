/config :
	config.php - массив для шаблонов , константы папок

/engine:
	file.php - для работы с файлами которые загружают
	page_func.php - для отображение данный на шаблоне 

/templates:
	index.html - главная страница + галерея
	down_foto.html - страница для загруски фото на сервер
	404.html - если пользователь запрашивает шаблона которого нет выдает эту страницу

/htdocs:
	index.php - обработчик запросов пользователя на шаблоны
	uploadfoto.php - обработчик загружаемых файлов
