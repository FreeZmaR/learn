
js: - директория с js файлами
 - chess.js  содержит класс ChessBoard .
    методы публичные:
        creatBoardAfter - создает доску после переданого элемента
        creatBoardIn - создает доску в переданом элементе
        setStyleCell - устонавливает ширину и высату клеткам
    не публичные :
        getCellName и numToLet - преобразуют координаты клетки (1А)
        jsonParser - переводит JSON в объект для экономии места (на будущее)
        activeCell - устонавливает на клетку слушателя который передает ее координаты (1А)
        build - компанует все элементы в один
 - otherWork.js содержит класс OtherWork для выполнение задач(1,2,3,5)
    функции публичные:
        show - вводет контент в DOM
    не публичные:
        colorReplace - запрашивает строгу ввиде аргумента, после возрощает удаляя не нужные символы
        findNum - выбирает числа из текста , с учетом десятичной дроби
        findClock - выбирает время из текста, есть ошибка не отлечит(2321:2312) получиться 21:23
        phoneNumber - контролирует коректность ввода телефонного номера, не дочет при исправление номера
        posport - контролирует правильный ввод паспортных данных, не дочет при редактирование

 - script.js содержит функции для управления DOM моделью
    функции:
        clear - удаляет всех детей у переданого элемента
        chois - обрабатывает выбор действий
        creatBoard - выводит доску в DOM
        menu - выводит кнопку назад
        chess_color - создает меню выборацвета фигур и передает в creatBoard


css: - директория со стилями

index.html - главная страница
chessPosition.json - хранит данные о фигурах (id,name,class,position,img)
